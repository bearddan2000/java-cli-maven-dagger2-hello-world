# java-cli-maven-dagger2-hello-world

## Description
A POC for dagger2 project.

## Tech stack
- java
- maven
  - dagger2
  - dagger2-compiler

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
